DISEÑO DE UNA PLATAFORMA PARA EL PROCESAMIENTO
DE DATOS ONLINE A PARTIR DE MUESTRAS BIOLÓGICAS
DE PACIENTES CON ERLIQUIOSIS CANINA

El sistema creado recopila, almacena, analiza y visualiza datos sobre la enfermedad en los caninos.

Funciones:

Recolección de datos:

Adquiere los datos generados a partir
de las muestras caninas tales como la saliva,
el pelo, etc.

Formulario de pacientes:

Es un documento para registrar
y almacenar información detallada
sobre el paciente.

Visualización de datos en tiempo real:

Permite al cliente monitorear
si los datos están funcionando correctamente.

Bases de datos:

Todos los datos recolectados
están encriptados lo cual ofrece
una mayor seguridad y privacidad.
